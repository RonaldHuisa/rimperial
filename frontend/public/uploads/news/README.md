# Imágenes de noticias Royal Imperial AI

Coloca aquí las imágenes que usarás en Noticias/Artículos.

Ruta pública de ejemplo para usar en el panel admin:

```txt
/uploads/news/noticia-bienvenida.webp
```

Recomendación:
- Usar `.webp` o `.jpg` comprimido.
- Ancho máximo recomendado: 1200px.
- Peso ideal: 100 KB a 400 KB por imagen.
- Evitar PNG pesados salvo que sea necesario.
